# AI Engineering Learning Path

A structured, 72-week roadmap focused on becoming an AI Engineer with strong foundations in Python, data pipelines, and LLM-based systems.

This is not a passive curriculum. Each phase is implementation-driven and designed to build production-relevant skills.

---

## Structure

The learning path is divided into 6 phases:

### Phase 1 — Python & Core Programming
- OOP, file I/O, modular design
- Writing clean, reusable code

### Phase 2 — Data Processing
- Pandas fundamentals
- Data transformation and cleaning
- Building small automation scripts

### Phase 3 — Data Pipelines
- Structured workflows
- Input → processing → output systems
- Basic pipeline design patterns

### Phase 4 — LLM Integration
- API-based model usage
- Prompt structuring
- Controlled generation

### Phase 5 — RAG Systems
- Retrieval pipelines
- Embeddings and vector databases
- Context-aware generation

### Phase 6 — Deployment & Systems
- Docker-based environments
- Simple production workflows
- System reliability basics

---

## Approach

- Weekly iterations with concrete outputs
- Focus on building systems, not just learning concepts
- Emphasis on:
  - clarity
  - reproducibility
  - real-world applicability
